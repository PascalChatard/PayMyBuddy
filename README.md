# PayMyBuddy
A money transfer app. 
This app uses Java to run and stores the data in MariaDB database.

## Technologies
> 1. Java 11
> 2. Spring 5.3.15.RELEASE
> 3. Spring Boot 2.6.3
> 4. JUnit 5
> 5. Maven 3.6.2
> 6. Eclipse 2018-12
